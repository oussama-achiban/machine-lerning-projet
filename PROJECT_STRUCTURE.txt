PIPELINE OVERVIEW
=================
1. Data loading and preprocessing
2. Dimensionality reduction (PCA, t-SNE, NMF)
3. Clustering (K-Means, Agglomerative, DBSCAN)
4. Classification (classical ML models)
5. Neural network training (PyTorch)
6. Evaluation and MLflow tracking
7. Reporting (notebook figures + LaTeX report)


KEY ENTRY POINTS
================
- Main pipeline: `python main.py`
- Notebook: `jupyter notebook notebooks/exploration.ipynb`
- MLflow UI: `mlflow ui --backend-store-uri file:./mlruns`
- Report build: `cd reports && pdflatex report.tex`
